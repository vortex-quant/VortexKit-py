# Contributing to VortexKit

Thank you for your interest in contributing to VortexKit! This document provides guidelines and standards for contributing to this library.

## About This Project

VortexKit is a Python reimplementation of the R `highfrequency` package, providing validated algorithms for statistical analysis of high-frequency financial data.
### Reference Implementation

### AI-Assisted Development

This project was developed with assistance from AI models, following validation protocols:
- All algorithms validated against reference R implementations
- Numerical accuracy verified to machine precision
- Edge cases tested against academic literature
- Comprehensive documentation and examples.

Contributors are welcome to use AI assistants when contributing, provided all submissions meet the accuracy and quality standards outlined below.


## Contribution Guidelines

### Types of Contributions

We welcome:
- **Bug fixes**: Correctness issues, edge cases, numerical stability
- **New algorithms**: Additional functions and algorithms.
- **Performance improvements**: Faster Numba implementations
- **Documentation**: Examples, tutorials, docstring improvements
- **Tests**: Additional test coverage, property-based tests


### Academic References

All implemented methods must include full citations:

```
References
----------
Barndorff-Nielsen, O.E. and Shephard, N. (2004).
```
